McDonald's - RPA

O projeto do McDonald's com a Dextra envolve o desenvolvimento de RPA's (Robot Process Automation), e a extração de dados do sistema NP6 da Fujitsu e disponibilização através de API's que são consumidas por sistemas dos franqueados.

Méritos
• Apredizado da linguagem GO para entregar 3 RPA's dois meses antes do prazo estabelecido.
• Unificação de uma infraestrutura para ETL que estava sendo repetida entre projetos.
• Desenvolvimentos rápidos (com qualidade) de customizações necessárias para o cliente no dia-a-dia.
• Criação de ferramentas para depuração de problemas no processo de ETL.
